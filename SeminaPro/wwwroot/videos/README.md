# Dossier Vidéos - SeminaPro

Ce dossier contient les vidéos de présentation de la plateforme SeminaPro.

## Format acceptés:
- .mp4
- .webm
- .ogg

## Exemple d'utilisation:

```html
<video width="100%" height="auto" controls>
  <source src="/videos/demo.mp4" type="video/mp4">
  Votre navigateur ne supporte pas la vidéo HTML5.
</video>
```

## Ajouter une vidéo:

1. Placez votre fichier vidéo dans ce dossier
2. Mettez à jour le chemin dans Index.cshtml
3. Ajustez la résolution si nécessaire
